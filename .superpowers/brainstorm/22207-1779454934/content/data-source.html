Opción A
Manual + Curado
Vos cargás los precios por zona desde fuentes confiables (Zonaprop, REMAX, CUCICBA). Panel de admin en la app para actualizar valores.
  • Listo en días, no semanas
  • Datos limpios y verificados
  • Control total sobre la calidad
  • Requiere actualización periódica tuya
  • No escala sin esfuerzo manual
Esfuerzo: Bajo · Velocidad: Alta
Automático
Opción B
Scraping automático
Cron jobs que extraen precios de Zonaprop, Argenprop y MercadoLibre Inmuebles. Datos frescos sin intervención manual.
  • Actualización automática continua
  • Miles de datos reales del mercado
  • Base para análisis de tendencias
  • 3–4 semanas de desarrollo
  • Los portales pueden bloquear el scraping
  • Datos crudos requieren limpieza
Esfuerzo: Alto · Velocidad: Baja
Híbrido
Opción C
Manual ahora, auto después
Arrancás con datos manuales para tener el sistema funcionando rápido. El scraping se agrega en una segunda fase cuando el core esté probado.
  • MVP en días
  • Arquitectura preparada para el auto
  • Sin riesgo de datos sucios al lanzar
  • Dos fases de desarrollo
  • Carga manual transitoria
Esfuerzo: Medio · Velocidad: Media
